üß†Leap Year Rules

The 4 Rules to Check:

Rule 1: Divisible by 400? ‚Üí LEAP YEAR ‚úÖ
if (Year % 400 == 0) return true;
Examples: 1600, 2000, 2400
Rule 2: Divisible by 100? ‚Üí NOT LEAP ‚ùå
else if (Year % 100 == 0) return false;
Examples: 1700, 1800, 1900, 2100
Rule 3: Divisible by 4? ‚Üí LEAP YEAR ‚úÖ
else if (Year % 4 == 0) return true;
Examples: 2004, 2008, 2012, 2016, 2020, 2024
Rule 4: All other years ‚Üí NOT LEAP ‚ùå
else return false;
Examples: 2001, 2002, 2003, 2005
üí° Important: The order matters! Always check 400 first, then 100, then 4. This is because 400 is divisible by both 100 and 4, so we need to catch it first!

üåçWhy Leap Years?

The Earth Problem:
Earth takes approximately 365.25 days to orbit the sun, not exactly 365 days!

The Solution:

1 Add 1 extra day every 4 years (0.25 √ó 4 = 1 day)
‚Üí This is why we have leap years!
2 But that's too much! (365.25 is not exact)
‚Üí Remove 3 days every 400 years
3 How? Skip leap years on century years (100, 200, 300...)
‚Üí Except every 400 years!

üéØ Quick Memory Trick:

  • 400? Always leap! ‚úÖ
  • 100? Never leap! ‚ùå
  • 4? Usually leap! ‚úÖ
  • Other? Not leap! ‚ùå

// Problem #2: Leap Year
#include <iostream>
#include <string>
using namespace std;

bool IsLeapYear(short Year)
{
    // leap year if perfectly divisible by 400
    if (Year % 400 == 0)
    {
        return true;
    }
    // not a leap year if divisible by 100
    // but not divisible by 400
    else if (Year % 100 == 0)
    {
        return false;
    }
    // leap year if not divisible by 100
    // but divisible by 4
    else if (Year % 4 == 0)
    {
        return true;
    }
    // all other years are not leap years
    else
    {
        return false;
    }
}

üîç Logic Breakdown:

üß†How We Got the One Line

Step-by-Step Optimization:

1 Identify when Year is Leap:
‚Ä¢ Divisible by 400 ‚Üí LEAP ‚úÖ
‚Ä¢ Divisible by 4 BUT NOT by 100 ‚Üí LEAP ‚úÖ
2 Write First Condition:
Year % 400 == 0
This catches: 1600, 2000, 2400...
3 Write Second Condition:
Year % 100 != 0 && Year % 4 == 0
This catches: 2004, 2008, 2024... (but NOT 1900, 2100)
4 Combine with OR:
return (Year % 400 == 0) || (Year % 100 != 0 && Year % 4 == 0);
üí° Key Insight: We use || (OR) because if EITHER condition is true, it's a leap year!

üîßLogical Operators

Understanding && and ||:

AND (&&) - Both Must Be True:
A && B ‚Üí true only if A is true AND B is true
Example: Year % 100 != 0 && Year % 4 == 0
Year 2024: NOT divisible by 100 ‚úÖ AND divisible by 4 ‚úÖ ‚Üí TRUE
Year 1900: Divisible by 100 ‚ùå ‚Üí FALSE (doesn't matter if divisible by 4)
OR (||) - At Least One Must Be True:
A || B ‚Üí true if A is true OR B is true (or both)
Example: (Year % 400 == 0) || (Year % 100 != 0 && Year % 4 == 0)
Year 2000: Divisible by 400 ‚úÖ ‚Üí TRUE (doesn't check second condition)
Year 2024: First condition ‚ùå but second condition ‚úÖ ‚Üí TRUE
NOT (!) - Flips the Value:
!A ‚Üí true if A is false, false if A is true
Note: We used != instead of ! for "not equal"

üéØ Operator Precedence:

  • && has higher precedence than ||
  • So: A || B && C is read as A || (B && C)
  • But it's better to use parentheses for clarity!

üî¢Why Different Data Types?

Understanding short vs int:

short (2 bytes) - Range: -32,768 to 32,767
‚úÖ Used for: Days and Hours
‚Ä¢ Days in year: 365 or 366 ‚úÖ
‚Ä¢ Hours in year: 8,760 or 8,784 ‚úÖ
Both fit easily in short!
int (4 bytes) - Range: -2,147,483,648 to 2,147,483,647
‚úÖ Used for: Minutes and Seconds
‚Ä¢ Minutes in year: ~525,600 or ~527,040 ‚ö†Ô∏è Too big for short!
‚Ä¢ Seconds in year: ~31,536,000 or ~31,622,400 ‚ö†Ô∏è WAY too big for short!
Must use int!

üéì Programming Best Practice:

  • Use the smallest data type that fits your data
  • Saves memory when dealing with large arrays
  • But don't go too small - causes overflow errors!
  • When in doubt, use int for safety

‚ùìTernary Operator (?:)

Short-Hand if-else

1 The Ternary Operator:
condition ? valueIfTrue : valueIfFalse;
2 Example from our code:
return IsLeapYear(Year) ? 366 : 365;
Reads as: "If IsLeapYear is true, return 366, otherwise return 365"
3 Same thing with regular if-else:
if (IsLeapYear(Year)) return 366; else return 365;
‚ú® Advantage: More concise! One line instead of 4!
